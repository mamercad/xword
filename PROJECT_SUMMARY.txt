QUICK START
================================================================================

1. Install from source:
   git clone https://github.com/mamercad/xword.git
   cd xword
   python3 -m pip install -e .

2. View a puzzle:
   xword show examples/simple.md

3. Create your own puzzle:
   Create my_puzzle.md with Markdown format
   xword show my_puzzle.md

4. Start a server:
   xword serve --port 8000

See QUICKSTART.md for more details.

KEY INSIGHTS & DESIGN DECISIONS
================================================================================

1. Markdown Format
   - Simple text format readable without tools
   - Works with version control (git diffs)
   - No external dependencies needed
   - Extensible with metadata (planned)

2. Sparse Grid Representation
   - Dictionary-based for memory efficiency
   - O(1) cell lookup performance
   - Scales to large puzzles
   - Easy metadata attachment

3. Modular Architecture
   - Core engine independent of UI
   - Multiple frontends can share engine
   - Testable components
   - Future plugin system ready

4. Session-Based Gameplay
   - Supports single and multiplayer
   - Tracks participant state
   - Time-aware for scoring
   - Serializable for storage

5. Async-Ready
   - FastAPI supports async
   - WebSocket infrastructure ready
   - Scalable to many concurrent games
   - Future mobile app compatible
